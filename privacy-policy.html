Privacy Policy

Last updated: 15 January 2026

1. Introduction

This application is a personal automation tool used to notify the user about work schedule updates. The application operates in accordance with the Facebook Platform Policies and is intended for personal use only.

2. Data Processed

The application processes the following data:

  • Text content of posts published in a Facebook group where the user is a member.

The application does not access:

  • Private messages
  • User profiles
  • Photos, videos, or comments
  • Any data outside the group content explicitly processed

3. Purpose of Data Processing

The data is processed solely to send SMS notifications to the user when new posts related to work schedule updates are published.

4. Data Storage and Retention

The application does not permanently store any personal data. Post content is processed temporarily and deleted immediately after SMS delivery.

5. Data Sharing

Data is not shared with third parties, except for an SMS delivery provider used exclusively to send notifications to the user.

6. User Scope

The application is intended for a single user and personal, non-commercial use.

7. Data Deletion

Since no data is stored, no data deletion request is required. The user can stop all data processing at any time by disabling the application.

8. Compliance with Facebook Policies

The application complies with Facebook Platform Policies and only processes data required for its stated functionality.
